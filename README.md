# vue-movie

> 一个电影 SPA，电影数据来自豆瓣，使用 travis CI 自动部署。 https://proj.vdorchan.com/vMovie

## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 + axios + sass + flex + svg + travis
